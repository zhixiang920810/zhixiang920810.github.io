# JDK与JRE的区别是什么

### JDK

JDK全称：Java Development Kit，是整个Java的核心，包含了Java运行环境（JRE）和一系列Java开发工具完整的包。

### JRE

JRE全称：Java Runtime Enviroment，是Java程序的运行环境，包含JVM、Java核心类库等。

JRE只能用来运行Java应用程序，不能用于变异开发，它是JDK的子集。
