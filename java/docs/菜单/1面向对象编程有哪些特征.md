# 面向对象编程有哪些特征

### 1. 封装（Encapsulation）

封装就是指隐藏对象的属性和实现细节，控制成员属性的访问和修改权限，通过特定公开的方法暴露给外面访问。

**比如&#58;**

```
	private String name = "天天刷、反复刷";
	
	public String getName(){
		return this.name;
	}
```

### 2. 继承（Inheritance）

继承就是指子类继承父类的成员和方法，使得子类也能具有父类相同的行为。

Java类智能实现单继承。

### 3. 多态（Polymorphism）

多态是指同一个行为具有多个不同的表现形式。

比如：方法重载，同一个方法名可以有不同参数的表现形式。
